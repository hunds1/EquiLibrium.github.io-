# Сайт конного клуба «Эквитас»

## Техническое задание

### Цель
Привлечение новых клиентов (взрослых и детей) на занятия верховой ездой, конные прогулки и мероприятия. Сбор заявок через звонки и формы.

## Структура сайта (блоки, сверху вниз)

### Шапка (Header)
- Логотип клуба (стилизованная голова лошади или всадника, название).
- Навигация: Главная, О нас, Услуги, Тренеры, Отзывы, Контакты.
- Телефон для записи (крупно, кликабельный).
- Кнопка «Записаться на пробное занятие».

### Главный экран (Hero-секция)
- Фон: Красивый, атмосферный баннер (всадник в лесу, лошадь крупным планом, закат в конюшне).
- Крупный заголовок: «Школа верховой езды [Название клуба]».
- Подзаголовок: «Откройте мир общения с лошадьми. Обучение для взрослых и детей, конные прогулки, иппотерапия».
- Кнопка «Выбрать занятие» (ведет к блоку с услугами).
- Кнопка «Записаться» (открывает форму).

### Блок «О нас» / Наша философия
- Краткое описание клуба: атмосфера, безопасность, любовь к животным.
- Ключевые факты в иконках:
  - Безопасность (шлемы, инструктаж).
  - Опытные тренеры.
  - Ухоженные лошади.
  - Индивидуальный подход.
- Фотогалерея (3-4 фото): уютная конюшня, занятия, прогулки на природе.

### Блок «Услуги»
- Карточки услуг в 2-3 столбца.
- Варианты услуг:
  - Индивидуальное занятие для новичков (с тренером в манеже). Цена. Кнопка «Записаться».
  - Групповые занятия для детей (секция). Цена. Кнопка «Записаться».
  - Конная прогулка по лесу (маршрут 1-2 часа). Цена. Кнопка «Узнать расписание».
  - Курс иппотерапии (коротко о пользе). Цена. Кнопка «Консультация».
- В каждой карточке: фото, название, краткое описание, цена, кнопка.

### Блок «Наши лошади»
- Красивый крупный фон (фото любимой лошади клуба).
- Текст рядом: «Знакомьтесь, это [Имя лошади, например, «Буян» или «Звездочка»]». Описание характера, породы, для кого подходит.
- Акцент на том, что все лошади здоровы, ухожены и подобраны под разный уровень всадников.

### Блок «Наши тренеры»
- Заголовок: «Наши инструкторы».
- Фотографии тренеров (в амуниции, с лошадьми).
- Имя, квалификация (опыт, разряд), специализация (дети, выездка, конкур).

### Блок «Акция» / «Спецпредложение»
- Фон: Романтичное фото (прогулка на закате, семейное занятие).
- Текст: «Подарочный сертификат на конную прогулку» или «Первое пробное занятие со скидкой 30%».
- Таймер обратного отсчета до конца акции (опционально).
- Форма для быстрой записи на акционное предложение (Имя + Телефон).

### Блок «Отзывы»
- Реальные отзывы клиентов (можно начать с тестовых).
- Фото автора отзыва (или аватарка).
- Текст отзыва о впечатлениях, прогрессе, атмосфере.

### Блок «Контакты и карта»
- Заголовок: «Как нас найти».
- Контактная информация:
  - Адрес конюшни.
  - Телефон для записи.
  - Email.
  - График работы (например, «Ежедневно с 9:00 до 20:00»).
- Интерактивная карта Яндекс/Google с отметкой локации.
- Краткие правила посещения (удобная одежда, обувь).

### Подвал (Footer)
- Уменьшенное лого, краткий слоган.
- Повтор основных контактов.
- Ссылки на соцсети клуба (Instagram, VK).
- Копирайт и ссылка на политику конфиденциальности.

## Технические требования
- Технологии: HTML5, CSS3 (Flexbox/Grid для адаптивности), JavaScript (для форм, таймера, модальных окон, мобильного меню).
- Адаптивность: Обязательна корректная работа на мобильных устройствах (адаптивное меню-бургер, перестроение блоков).
- Формы: Валидация (имя, телефон, email), отправка на email/Telegram через PHP-обработчик или сервис (Formspree, Telegram Bot).
- Оптимизация: Сжатие изображений, ленивая загрузка (lazy load).
- Хостинг: Можно использовать GitHub Pages, Vercel, Netlify (для статики) или недорогой виртуальный хостинг с поддержкой PHP.

## Содержание (копирайт) для наполнения
- Нужно будет придумать:
  - Название клуба
  - Слоганы и тексты для каждого блока (атмосферные, про дружбу с лошадьми, про безопасность и пользу).
  - Имена и биографии для тренеров и лошадей.
  - Описание услуг с понятными преимуществами.
